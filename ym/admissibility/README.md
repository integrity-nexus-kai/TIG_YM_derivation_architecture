This directory contains admissibility structures for theorem-oriented Yang–Mills infrared dynamics.

- exclusion of stable scale-invariant sectors
- operator-compatible admissibility
- finite-correlation admissibility
- asymptotic consistency conditions
- spectral admissibility constraints

Goal:
to formulate mathematically controlled admissibility criteria compatible with nonlinear infrared instability.

Status:
proof-oriented / conjectural / not yet rigorous.
