# TIG3 Theorem Candidates

## Purpose

This document collects the current preliminary theorem targets of the TIG3 admissibility program.

The statements listed here are not established results.

They define candidate directions for future rigorous derivation within the TIG3 vacuum-sector framework.

The purpose of this document is organizational and structural.

---

# Foundational Strategy

The TIG3 theorem program investigates whether the TIG2 branch structure may admit:

- admissible vacuum continuation,
- bounded high-curvature sectors,
- structurally stable near-critical dynamics,
- and asymptotic GR-compatible regulated geometries.

The theorem candidates therefore remain conditional upon explicit assumptions.

---

# Candidate Theorem 1 — Admissible Branch Continuation

## Informal Statement

Under suitable admissibility assumptions, the TIG2 branch structure admits geometrically regular vacuum continuation.

## Structural Targets

Candidate properties include:

- Lorentzian metric regularity,
- admissible branch continuation,
- and controlled near-critical deformation.

## Current Status

Partially conjectural.

No rigorous proof currently exists.

# Candidate Theorem 2 — Bounded Curvature Sector

## Informal Statement

There exist admissible TIG vacuum sectors in which relevant geometric invariants remain finite throughout the regulated core regime.

## Candidate Bounded Quantities

Potentially bounded invariants include:

- Kretschmann scalar,
- Ricci invariants,
- and admissible higher-order curvature structures.

## Current Status

Heuristic and unproven.

Requires explicit geometric control.

# Candidate Theorem 3 — Near-Critical Structural Stability

## Informal Statement

Near the critical branch regime

## Candidate Structural Features

Potential phenomena include:

- metastable geometric sectors,
- bounded relaxation modes,
- near-critical stabilization,
- and controlled curvature amplification.

## Current Status

Currently conjectural.

Supported only heuristically through branch analysis and exploratory simulation.

## Informal Statement

- controlled branch deformation,

beta -> 0

## Current Status

Expected structural requirement but not yet rigorously established.

# Candidate Theorem 5 — Structural Admissibility Constraint

## Informal Statement

Admissible vacuum continuation imposes nontrivial structural constraints on allowable branch geometries.

## Interpretation

Not every mathematically possible continuation is admissible.

The admissible class is constrained simultaneously by:

- branch consistency,
- bounded curvature behavior,

Programmatic theorem target.

Not yet formalized.

# Excluded Claims

The TIG3 theorem program does not currently claim:

- complete singularity elimination,
- quantum-gravity completion,
- observational verification,
- or a replacement of general relativity.

The framework remains strictly structural and geometric.

The TIG3 theorem program investigates whether admissible branch geometry naturally constrains the collapse sector into dynamically regulated continuation.

- admissibility,
- bounded continuation,
- structural regularization,
- and near-critical branch dynamics,

within a conservative geometric framework.
