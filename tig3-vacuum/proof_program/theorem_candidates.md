# TIG3 Theorem Candidates

## Purpose

This document collects preliminary theorem targets for the TIG3 admissible vacuum program.

## Candidate Directions

### 1. Admissible Branch Regularity

Under suitable admissibility assumptions, the TIG2 branch sector admits bounded geometric continuation.

### 2. Finite Curvature Sector

There exist admissible parameter sectors in which relevant curvature invariants remain finite.

### 3. Near-Critical Stability

Near the TIG2 critical point, admissible branch configurations exhibit dynamically bounded behavior.

### 4. GR-Compatible Regularization

The regulated TIG3 sector continuously reduces to standard GR behavior away from the critical regime.

## Status

These are currently theorem candidates only and not formal proofs.
