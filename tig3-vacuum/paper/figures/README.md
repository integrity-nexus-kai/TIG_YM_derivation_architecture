# TIG3 Figure Architecture

## Purpose

This document defines the canonical publication figure structure for the TIG3 vacuum-sector manuscript.

Its purpose is:

- figure stabilization,
- visual consistency,
- publication-ready formatting,
- and prevention of interpretive drift in graphical material.

---

# Figure Philosophy

All TIG3 figures must satisfy:

```text
mathematical interpretability
> visual spectacle.
```

Figures are intended to:

- clarify theorem structure,
- illustrate perturbative behavior,
- and support operator-theoretic interpretation.

Figures must not:

- imply unsupported physical claims,
- suggest observational confirmation,
- or introduce speculative metaphysical imagery.

# Figure Style Policy

All figures should:

- use publication-safe typography,
- maintain axis readability,
- avoid visual clutter,
- use mathematically meaningful scales,
- and remain grayscale-compatible when possible.

---

# Caption Policy

Every figure caption must:

- define all variables,
- avoid speculative interpretation,
- and distinguish between:
  - established,
  - conditional,
  - and conjectural
  structures.

---

# Forbidden Figure Elements

The following are prohibited:

- artistic black-hole imagery,
- glowing spacetime visuals,
- quantum vortex imagery,
- undefined dimensions,
- cosmological metaphors,
- AI-generated speculative physics art.

Figures must remain mathematically disciplined.
