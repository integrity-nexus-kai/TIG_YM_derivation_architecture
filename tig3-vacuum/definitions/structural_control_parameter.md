# Structural Control Parameter

## Purpose

This document defines the role of the TIG structural parameter beta within the TIG2/TIG3 branch sector.

## Definition

The TIG structural control parameter is defined as

beta = r_c / (2M)

where:

- r_c denotes the structural core scale,
- M denotes the asymptotic mass scale.

## Interpretation

Within TIG3, beta is treated as:

- a dimensionless structural control parameter,
- a branch-sector regulator,
- a geometric scaling quantity.

It is not interpreted as:

- an additional spatial dimension,
- a hidden coordinate,
- or a free metaphysical degree of freedom.

## Role in TIG2

The parameter beta governs the cubic branch structure

x^3 - x^2 + beta^3 = 0

including:

- branch admissibility,
- discriminant structure,
- saddle-node criticality,
- and near-critical scaling.
