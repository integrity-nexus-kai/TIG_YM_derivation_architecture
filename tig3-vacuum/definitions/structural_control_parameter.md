# Structural Control Parameter

## Purpose

This document defines the role of the TIG structural parameter beta within the TIG2/TIG3 admissibility framework.

The purpose of the parameter is not to introduce an additional dimension or hidden coordinate structure, but to characterize admissible geometric deformation of the TIG horizon branch sector.

---

# Definition

The TIG structural control parameter is defined as

beta = r_c / (2M)

where:

- r_c denotes the characteristic structural core scale,
- M denotes the asymptotic mass scale.

The parameter beta is dimensionless.

---

# Role in TIG2

Within TIG2, beta governs the cubic horizon branch structure

x^3 - x^2 + beta^3 = 0

whose discriminant structure determines:

- admissible horizon sectors,
- branch merger behavior,
- critical transitions,
- and saddle-node scaling.

The critical value

beta_c^3 = 4/27

marks the degeneracy point of the admissible horizon branch structure.

---

# Structural Interpretation

Within TIG3, beta is interpreted as a structural deformation parameter controlling the admissible vacuum geometry.

In particular:

- increasing beta modifies the admissible branch sector,
- near-critical beta produces enhanced structural sensitivity,
- the branch geometry changes continuously with beta,
- and admissibility conditions become increasingly restrictive near the critical regime.

The parameter therefore acts as a geometric control parameter for the TIG vacuum sector.

---

# Restrictions

Within the TIG framework, beta is explicitly not interpreted as:

- an additional spatial dimension,
- a hidden physical coordinate,
- a metaphysical variable,
- or an unconstrained phenomenological parameter.

Its role is strictly structural.

---

# Near-Critical Regime

The regime

beta ≈ beta_c

is structurally distinguished by:

- branch merger behavior,
- enhanced sensitivity of the horizon structure,
- near-critical scaling,
- and possible emergence of dynamically regulated high-curvature sectors.

The detailed behavior of this regime remains part of the TIG3 proof program.

---

# GR Limit

The limit

beta -> 0

must continuously recover standard Schwarzschild-compatible behavior away from the regulated core sector.

This requirement is treated as a non-negotiable consistency condition.
