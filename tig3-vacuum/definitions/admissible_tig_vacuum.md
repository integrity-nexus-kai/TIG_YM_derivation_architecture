# Admissible TIG Vacuum Configuration

## Definition

An admissible TIG vacuum configuration is a spacetime geometry satisfying:

1. Lorentzian metric regularity,
2. asymptotic compatibility with the GR limit,
3. finite admissible curvature invariants,
4. compatibility with the TIG2 horizon branch structure,
5. stable structural parameter continuation,
6. exclusion of singular branch degeneration.

The admissibility problem is the central entry point into TIG3.
