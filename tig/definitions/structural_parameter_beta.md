# Definition

The TIG structural parameter is defined as

\[
\beta := \frac{r_c}{2M},
\]

where:

- \(M\) denotes the effective gravitational mass scale,
- \(r_c\) denotes the structural correction scale.

\[
\beta

# Role in TIG

The parameter enters the structural horizon equation

\[
x^3 - x^2 + \beta^3 = 0,

\[
x := \frac{r_H}{2M}.
\]

The parameter controls the deviation from the Schwarzschild reference geometry.

# Schwarzschild Limit

The Schwarzschild configuration is recovered for

\[
\beta \rightarrow 0.
\]

In this limit the structural equation reduces to

\[
x^2(x-1)=0.
\]

The admissible outer horizon branch satisfies

\[
x \rightarrow 1.
\]
