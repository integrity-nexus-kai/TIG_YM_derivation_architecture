# Canonical Horizon Equation

The canonical TIG horizon equation is:

\[
x^3-x^2+\beta^3=0.
\]

---

# Critical Limit

The critical limit is defined by:

\[
\beta \rightarrow \beta_c
\]

with:

\[
\beta_c^3=\frac{4}{27}.
\]

At this value the discriminant satisfies:

\[
\Delta=\beta^3(4-27\beta^3)=0.
\]

---

# Critical Horizon State

The critical branch point occurs at:

\[
x_c=\frac{2}{3}.
\]

This follows from simultaneous satisfaction of:

\[
P(x,\beta)=0
\]

and:

\[
\frac{\partial P}{\partial x}=0.
\]

---

# Structural Interpretation

As:

\[
\beta \rightarrow \beta_c
\]

the admissible horizon branch approaches a degenerate critical configuration.

This behavior currently represents:

- branch coalescence,
- discriminant collapse,
- restricted structural accessibility,
- and admissibility-compatible critical compression.

# Perturbative Limitation

Near the critical point ordinary branch perturbation becomes structurally fragile.

This is because:

- root separation decreases,
- branch sensitivity increases,
- and local linearization becomes insufficient.

No complete perturbative theorem is currently implied.

# Admissibility Requirement

An admissible critical-limit behavior must preserve:

- real positive horizon structure,
- branch continuity where defined,
- bounded structural interpretation,
- and compatibility with the critical discriminant.

Branches violating these requirements are non-admissible.
