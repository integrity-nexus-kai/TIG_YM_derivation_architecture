# Core Principle

\[

\]

---

# Canonical Horizon Equation

The canonical TIG horizon equation is:

\[
x^3-x^2+\beta^3=0.
\]

The admissible outer branch is denoted by:

\[
x=x_+(\beta).
\]

---

# Schwarzschild Limit

For:

\[
\beta \rightarrow 0
\]

the admissible outer branch satisfies:

\[
x_+(\beta)\rightarrow 1.
\]

This preserves compatibility with the Schwarzschild-connected sector.

# Critical Limit

For:

\[
\beta \rightarrow \beta_c
\]

with:

\[
\beta_c^3=\frac{4}{27},
\]

the admissible outer branch approaches:

\[
x_+\rightarrow \frac{2}{3}.
\]

# Monotonicity Interpretation

Within the admissible regime:

\[
0\leq \beta \leq \beta_c,
\]

the outer branch decreases continuously from:

\[
x=1
\]

toward:

\[
x=\frac{2}{3}.
\]

This behavior currently represents:

- admissibility-compatible branch compression,
- reduced horizon accessibility,
- and critical structural restriction.
