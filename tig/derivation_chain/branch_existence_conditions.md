# Core Principle

Within TIG:

\[
\text{admissible horizon branches exist only under discriminant-compatible conditions}
\]

Branch existence is therefore structurally constrained by the canonical horizon equation and admissibility requirements.

---

# Canonical Horizon Equation

The canonical TIG horizon equation is:

\[
x^3-x^2+\beta^3=0.
\]

Admissible horizon branches correspond to admissible real positive roots of this equation.

---

# Discriminant Requirement

The associated discriminant is:

\[
\Delta=\beta^3(4-27\beta^3).
\]

Admissible branch existence requires:

\[
\Delta \geq 0.
\]

Thus:

\[
\beta^3 \leq \frac{4}{27}.
\]

---

# Critical Existence Boundary

The critical branch boundary occurs at:

\[
\beta_c^3=\frac{4}{27}.
\]

At this value the admissible branch structure becomes degenerate.

For:

\[
\beta^3>\frac{4}{27}
\]

the canonical branch structure loses admissible real positive continuation.

---

# Admissible Root Conditions

A structurally admissible branch must satisfy:

- real-valuedness,
- positivity,
- branch continuity,
- asymptotic compatibility,
- and discriminant consistency.

Roots violating these conditions are non-admissible.

---

# Schwarzschild Compatibility

For:

\[
\beta \rightarrow 0
\]

the admissible branch structure must continuously recover the Schwarzschild-compatible limit:

\[
x \rightarrow 1.
\]

This condition preserves low-curvature compatibility.
