# First Derivative Structure

The radial derivative of the lapse function is:

\[
F'(r)
=
-\frac{d}{dr}
\left(
\frac{2Mr^2}{r^3+r_c^3}
\right).
\]

Applying the quotient rule gives:

\[
F'(r)
=
-\frac{
4Mr(r^3+r_c^3)
-
6Mr^4
}{
(r^3+r_c^3)^2
}.
\]

Simplifying:

\[
F'(r)
=
\frac{
2Mr^4
-
4Mr\,r_c^3
}{
(r^3+r_c^3)^2
}.
\]
