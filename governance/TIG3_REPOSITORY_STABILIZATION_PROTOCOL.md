# 3. Mandatory Classification System

Every research document must contain one explicit classification label.

Allowed labels:

```text
[canonical]
[exploratory]
[heuristic]
[speculative]
[deprecated]
```

Definitions:

| Classification | Meaning |
|---|---|
| canonical | stabilized repository core |
| exploratory | mathematically plausible but incomplete |
| heuristic | conceptual or analogical reasoning |
| speculative | unsupported structural hypothesis |
| deprecated | rejected or unstable structure |

Unclassified files are considered unstable by default.
