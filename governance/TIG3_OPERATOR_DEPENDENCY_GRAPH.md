# 1. Purpose

This document defines the dependency structure of the TIG3 operator-theoretic architecture.

The purpose is to:

- prevent circular reasoning,
- stabilize theorem ordering,
- clarify prerequisite structures,
- preserve operator-theoretic consistency,
- maintain admissibility-controlled mathematical development.

The repository remains exploratory.

No Yang--Mills Mass Gap proof is claimed.

---

# 2. Core Dependency Principle

```text
later-stage spectral claims require
earlier-stage operator control
```

Thus:

- spectral conclusions require operator realization,
- operator realization requires closed quadratic forms,
- quadratic forms require admissible domains,
- admissible domains require admissibility structure.

---

# 3. Canonical Dependency Chain

```text
admissibility
→ admissible domains
→ quadratic forms
→ closability
→ semiboundedness
→ self-adjointness
→ semigroup evolution
→ compactness mechanisms
→ compact resolvent structure
→ spectral stabilization
→ conditional spectral separation
```

# 9. Self-Adjointness Layer

Self-adjointness requires:

- densely defined operators,
- symmetric structure,
- closed or closable forms,
- domain compatibility,
- boundary-condition control.

Potential frameworks:

- Friedrichs extension,
- Kato representation theory.

Dependency:

```text
self-adjointness depends on semibounded closable forms
```

---

# 10. Semigroup Layer

Semigroup evolution requires:

- self-adjoint or dissipative structure,
- strong continuity,
- admissibility-preserving flow.

- Hille--Yosida theory.

Dependency:

```text
semigroup evolution depends on operator realization
```

compact resolvent structure depends on compactness mechanisms

# 13. Spectral-Stabilization Layer

Spectral stabilization requires:

spectral stabilization depends on compact resolvent structure

# 14. Conditional Spectral Separation Layer

Conditional spectral separation requires:

- isolated admissible spectral sectors,
- perturbative spectral persistence,
- essential-spectrum control.

Dependency:

```text
conditional spectral separation
depends on spectral stabilization
```

This is not equivalent to a Yang--Mills Mass Gap proof.

# 15. Dependency-Control Principle

Every new document should explicitly state:

1. which earlier layers it depends on,
2. which assumptions are inherited,
3. which structures remain unresolved,
4. whether the document is:
   - rigorous,
   - exploratory,
   - heuristic,
   - speculative.
