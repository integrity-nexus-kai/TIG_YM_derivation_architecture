## 3.1 Quadratic-form sector

Suppose TIG3 defines a perturbative admissibility functional:

\[
Q_{\mathrm{TIG}}[\psi]
=
\langle \psi, L_{\mathrm{TIG}}\psi\rangle .
\]

The central mathematical questions become:

1. Is \(Q_{\mathrm{TIG}}\) densely defined?
2. Is it symmetric?
3. Is it closable?
4. Is it bounded from below?
5. Does it define a self-adjoint operator?
6. Does it satisfy coercive estimates?

A representative coercive estimate would take the form:

\[
Q_{\mathrm{TIG}}[\psi]
\geq
c \|\psi\|_{\mathcal H_1}^{2}
-
C \|\psi\|_{\mathcal H}^{2},
\qquad c>0.
\]

If such estimates hold, then the TIG admissibility framework possesses mathematically meaningful operator structure.

This alone would still not imply a Yang--Mills mass gap.

## 3.2 Spectral admissibility

\[

\[
