# Purpose

This document defines the canonical minimal structural core of TIG.

Its purpose is to consolidate the current TIG architecture into a rigorously controlled foundational layer from which all higher admissibility and operator-oriented structures may evolve.

---

# Core Principle

TIG currently defines a structural admissibility framework for horizon-related asymptotic behavior.

TIG does NOT yet define:

- a complete field theory,
- a quantum gravity theory,
- a Yang–Mills proof,
- or a rigorous operator-theoretic completion.

---

The current TIG core consists only of the following primitive structural objects:

1. admissible horizon states
2. structural horizon branches
3. structural parameter beta
4. discriminant-controlled criticality
5. admissibility-compatible asymptotic restriction

No additional primitive objects are currently canonical.
