# 1. Purpose

This document defines the claim-classification audit for the TIG3 operator-theoretic research architecture.

The purpose is to:

- separate rigorous statements from exploratory structures,
- prevent theorem inflation,
- preserve mathematical honesty,
- stabilize publication discipline,
- maintain explicit distinction between:
  - established mathematics,
  - conditional structures,
  - heuristics,
  - speculative interpretation.

The repository remains exploratory.

No Yang--Mills Mass Gap proof is claimed.

# 3. Classification Categories

| Category | Meaning |
|---|---|
| rigorous | mathematically established |
| conditional | dependent on unresolved assumptions |
| exploratory | plausible but incomplete |
| heuristic | conceptual or analogical |
| speculative | unsupported or highly incomplete |
| deprecated | unstable or rejected |

1. mathematical definition status,
2. operator-theoretic completeness,
3. proof availability,
4. spectral-control status,
5. dependence on unresolved assumptions,
